## Getting Started

### For Developers

#### Step 1: Sign Up
```bash
# Register account
curl -X POST https://api.novacron.dev/auth/register \
  -d '{"email":"dev@example.com","name":"Developer"}'
```

#### Step 2: Start Learning
- Browse available labs
- Choose certification path
- Complete first lab
- Track progress

#### Step 3: Build Projects
- Select project
- Design solution
- Implement code
- Submit for review

#### Step 4: Get Certified
- Meet requirements
- Apply for certification
- Pass final assessment
- Receive certificate

### For App Developers

#### Step 1: Build App
```bash
# Initialize app
npx novacron-cli init my-app

# Develop locally
npx novacron-cli dev

# Test app
npx novacron-cli test
```

#### Step 2: Submit App
```bash
# Package app
npx novacron-cli package

# Submit for review
npx novacron-cli submit
```

#### Step 3: Publish
- Await approval
- Configure pricing
- Publish to marketplace

### For Contributors

#### Step 1: Find Issues
```bash
# List open issues
gh issue list --repo novacron/core --label "good-first-issue"
```

#### Step 2: Contribute
```bash
# Fork and clone
gh repo fork novacron/core
git clone https://github.com/YOUR_USERNAME/core

# Create branch
git checkout -b fix/issue-123

# Make changes
# Commit and push
git commit -am "Fix: Resolve issue #123"
git push origin fix/issue-123

# Create PR
gh pr create
```
