# Roadmap: NovaCron

## Overview

NovaCron is building a distributed L2 hypervisor platform with latency-based node clustering, LUN storage for efficient migration, and automatic failover. The journey takes the existing distributed systems foundation and adds the critical missing pieces for cross-internet VM operations.
