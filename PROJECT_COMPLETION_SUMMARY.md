## ğŸ“ Quick Start

### Installation
```bash
# Clone repository
git clone https://github.com/khryptorgraphics/novacron
cd novacron

# Install dependencies
npm install

# Start services
docker-compose up -d

# Run migrations
npm run migrate

# Start platform
npm run start
```

### Usage
```bash
# Start auto-spawning system
npm run auto-spawn:start

# Deploy VM
novacron vm create --cpu 4 --memory 8G --provider aws

# Migrate VM
novacron vm migrate <vm-id> --target azure

# Monitor system
novacron status --detailed
```
