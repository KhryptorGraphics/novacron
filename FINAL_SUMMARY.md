## ğŸš€ Quick Start

### Installation
```bash
# Clone repository
git clone https://github.com/khryptorgraphics/novacron
cd novacron

# Install dependencies
npm install

# Start services
docker-compose up -d

# Run migrations
npm run migrate

# Start platform
npm run start
```

### First VM
```bash
# Create VM
novacron vm create \
  --name web-server-1 \
  --provider aws \
  --cpu 4 \
  --memory 8G

# List VMs
novacron vm list

# Migrate VM
novacron vm migrate <vm-id> --target-provider azure
```
